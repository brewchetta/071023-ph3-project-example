# Cave Manager

Are you a neanderthal? Have you been edged out of the cave market, forced to live above ground in the forests of ancient France? Good news! This app has been developed specifically for you to manage and track potential caveside properties including a novel new concept, rocks as currency!

## Installation

Download the application and install with `pipenv install`. Enter your python shell with `pipenv shell`.

## Running the Application

In your primitive rock computer run `python3 app.py` to begin the application.
